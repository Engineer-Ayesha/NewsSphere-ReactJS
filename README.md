# 📰 NewsSphere – React.js News Application
NewsSphere is a responsive news application built with **React.js** that provides the latest headlines across multiple categories using the **NewsAPI**. The project offers a smooth user experience with infinite scrolling, loading indicators, and secure API integration using **Netlify Functions**.

## ⚙️ Installation
Clone the repository:
```bash
git clone https://github.com/Engineer-Ayesha/NewsSphere-ReactJS.git
```
Go to the project folder:
```bash
cd NewsSphere-ReactJS
```
Install dependencies:
```bash
npm install
```
Create a `.env.local` file:
```env
REACT_APP_API_NEWSSPHERE=YOUR_NEWSAPI_KEY
```
Run the application:
```bash
npm start
```

Open:

```
http://localhost:3000
```
## 🚀 Production Deployment
The application is deployed on **Netlify**.
To avoid browser CORS restrictions from NewsAPI, requests are securely handled through **Netlify Functions**, keeping the API key protected in production.
