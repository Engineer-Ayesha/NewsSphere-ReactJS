# NewsSphere
NewsSphere is a responsive React.js news website that displays the latest headlines using the NewsAPI.
## Features
- Latest news headlines
- Multiple news categories
  - General
  - Business
  - Entertainment
  - Health
  - Science
  - Sports
  - Technology
- Infinite scrolling
- Loading progress bar
- Responsive design
- React Router navigation
- Environment variables for API key security
## Technologies Used
- React.js
- React Router DOM
- Bootstrap
- NewsAPI
- React Top Loading Bar
- React Infinite Scroll Component
## Installation
```bash
npm install
npm start
```
## Environment Variables
Create a `.env.local` file in the project root and add:
```env
REACT_APP_API_NEWSSPHERE=YOUR_API_KEY
```
